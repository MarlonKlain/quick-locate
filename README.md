# QuickLocate

QuickLocate is a mobile application developed to assist in managing and locating products in warehouse environments. Built with practicality in mind, QuickLocate enables workers to easily check and update the physical location of items, ensuring better organization and reducing time lost in search processes.

---

## 🚀 Motivation
As someone who works in a warehouse, I frequently experienced the challenges of finding items without clear identification or tracking. This real-life need sparked the idea to create QuickLocate — an app designed to bring simple and efficient warehouse location management right into your hands.

QuickLocate also represents a major milestone in my journey as a developer. This project helped me consolidate skills in mobile development, backend implementation, and database handling.

---

## 📱 Project Overview

QuickLocate offers a streamlined solution to manage the location of warehouse items. It allows the user to import data from Excel files, view all products in a simple list, and update the location of each item as needed.

---

## ✅ Features

- 📂 **Import Data from Excel** – Load your warehouse data into the app directly from an exported Excel file.
- 🔍 **Product List with Filters** – Browse all products and filter results to find exactly what you need.
- ✏️ **Location Editing** – Update the physical location of any item in a simple and intuitive interface.
- 🔠 **Location Format Validation** – Ensures that new locations follow a specific 5-character format (e.g., `15AC2`).
- 📋 **Change Report** – Automatically generates a report summarizing changes made during the import process.
